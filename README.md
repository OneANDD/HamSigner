# HamSign

This website has everything you need to conqueror the sideloading community!

# Features
- Open source and free
- Sign, install, and have the option to modify the ipa
- Check certificate entitlements with .p12 or .mobileprovision
- Change p12 password to whatever your heart desires


# Privacy and Security

- All files are processed securely on our servers and deleted immediately after processing.
- Your certificates and provisioning profiles are never stored or logged.
